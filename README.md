## Description

Backend for scraping news websites, with sentiment analysis.

## Tech Stack

- Python
- Mongo
- Scrapy
- Nltk(sentiment analysis)
- TDD(pytest)
- Linting(pylint)

## How to Run IT

### Running Crawler with scrapy

    scrapy crawl article

### Testing With pytest

    python -m pytest tests/
